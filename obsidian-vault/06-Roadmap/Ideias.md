# Ideias

> Este projeto **não mantém um banco de ideias soltas** separado do backlog/roadmap. `docs/TASKS.md`
> tem uma seção "Notas livres" nomeada para esse propósito
> ("observações, decisões pendentes pequenas, ideias durante a implementação"), mas na prática
> ela hoje só registra itens já **concluídos** (correções de segurança, migração do SheetJS,
> lazy routes) — não ideias em aberto.

## Onde o pensamento exploratório realmente vive

- **Riscos do cronograma** (`docs/ROADMAP.md`) — hipóteses de risco com probabilidade/impacto/
  mitigação, algumas já resolvidas
- **Alternativas consideradas** dentro de cada ADR (`docs/decisions/`) — as opções descartadas e
  por quê, o mais próximo de um registro de "ideias avaliadas"
- **[[Backlog]]** — itens fora do escopo atual mas já reconhecidos como desejáveis

Se este projeto começar a acumular ideias soltas de verdade, esta nota é o lugar certo para
registrá-las — hoje ela reflete honestamente que essa prática ainda não existe.
